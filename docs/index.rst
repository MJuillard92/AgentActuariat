Agent Actuariel — Documentation
================================

Bienvenue dans la documentation de l'**Agent Actuariel**, un système automatisé
de construction de tables de mortalité d'expérience.

Cette documentation s'adresse aux actuaires qui utilisent l'outil au quotidien.
Elle explique comment le configurer, comment il fonctionne en coulisses, et comment
interpréter ses décisions.
